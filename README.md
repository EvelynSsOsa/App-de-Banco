# App-de-Banco
Pequeña aplicación web desarrollada con Streamlit que permite la gestión de cuentas, realizar depósitos y retiros, algunas de sus funciones son las siguientes: <br>

🌸Consultar historial de movimientos con fecha y hora.<br>
🌸Creación de usuarios<br>
🌸Retiro<br>
🌸Deposito<br>
🌸Consulta de saldo<br>
🌸Persistencia de datos JSON<br>

Estas fueron las tecnologias que utilice:<br><br>
🌺 Python<br>
🌺Streamlit<br>
🌺Pandas<br>
🌺JSON<br>
🌺Programación Orientada a Objetos (POO)<br><br>

Las características de mi aplicación son:<br>
🪷Interfaz moderna estilo fintech(Prototipo)<br>
🪷Diseño responsivo<br>
🪷Tarjetas visuales para mostrar información del usuario<br> 
🪷Formato de dinero tipo banco ($2,500.00)<br> 
🪷Inputs interactivos para ingreso de montos (depósitos y retiros)<br> 
🪷Manejo de estado con session_state<br>
🪷Persistencia de datos con archivos JSON<br><br> 

🌷La estructura del proyecto:<br><br>  
│── CuentaBanco.py<br>
│── CuentaBancoInterfaz.py<br>
│── usuarios.json<br>
│── .streamlit/config.toml<br>
│── README.md<br><br>
